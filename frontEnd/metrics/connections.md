# Connections - Meaningful Relationships

Log meaningful connections from content output.

## Format

```
Date | Platform | Person | Role | Context
```

## Template

```
YYYY-MM-DD | X/IG/YT/RN | Name | VC/Founder/Lead/Other | How we connected
```

## Examples

```
2026-02-25 | X | John Doe | VC @ Sequoia | Replied to post about agents
2026-02-20 | IG | Jane Smith | Founder @ AI Startup | DM about fitness discipline
```
