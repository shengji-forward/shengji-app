# IP Tracking - Original Ideas That Spread

Track original ideas and their propagation.

## Format

```
Idea | Origin Date | First Platform | Evidence of Spread | Notes
```

## Template

```
Idea description | YYYY-MM-DD | X/IG/YT/RN | Screenshots/links/references | Context
```

## Examples

```
"Productise yourself" | 2026-02-25 | X | @user referenced in reply | Core philosophy tweet
"Agents as services" | 2026-02-20 | X | 3 retweets with commentary | Product positioning
```
